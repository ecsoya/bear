# how to use

Step 1. Add the JitPack repository to your build file

```
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```

Step 2. Add the dependency

```
	<dependency>
	    <groupId>com.github.ecsoya.bear</groupId>
	    <artifactId>bear</artifactId>
	    <version>${release version}</version>
	</dependency>
```
